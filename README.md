# trend-radar

키워드 트렌드 모니터링 + 네이버 블로그 · 인스타그램 카드뉴스 자동 생성 도구.
설치 없이 브라우저에서 바로 동작하는 정적 웹앱입니다 (GitHub Pages 배포 가능).

## 구성

| 페이지 | 기능 |
|---|---|
| `index.html` — 트렌드 레이더 | 키워드 하나로 뉴스 · 유튜브 · 블로그/카페 반응을 한 번에 수집, 언급 추이 · 단어 빈도 분석 |
| `studio.html` — 콘텐츠 스튜디오 | 수집된 트렌드 데이터로 **네이버 블로그 글**과 **인스타 카드뉴스(1080×1080 PNG)** 를 자동 생성 |

## 사용 흐름 (자동화 파이프라인)

1. **트렌드 레이더**에서 키워드 검색 → 뉴스·커뮤니티 반응 자동 수집
2. 결과 아래 **"✨ 이 키워드로 블로그 글 + 인스타 카드뉴스 만들기"** 클릭
3. **콘텐츠 스튜디오**에서 ⚡ 자동 생성 버튼 클릭
   - 블로그: SEO 제목 후보 3개 + 본문 + 해시태그 → **서식 포함 복사** → 네이버 에디터에 붙여넣기 → 발행
   - 카드뉴스: 슬라이드 7장 자동 구성 → 템플릿/색상 선택 → **PNG 전체 다운로드** + 캡션 복사 → 인스타 업로드

### AI 연동 (선택)

콘텐츠 스튜디오의 "AI 연동 설정"에 API 키를 넣으면 글 품질이 크게 좋아집니다.

- **Google Gemini** — [Google AI Studio](https://aistudio.google.com/apikey)에서 무료 키 발급
- **Anthropic Claude** — [Anthropic Console](https://console.anthropic.com/)에서 키 발급

키가 없어도 트렌드 레이더에서 수집한 헤드라인으로 초안을 자동 조립합니다.
모든 키는 사용자 브라우저(localStorage)에만 저장되며 서버로 전송되지 않습니다.

### 트렌드 레이더 API 키 (선택)

- **유튜브 Data API v3** — 영상·조회수·댓글수 수집
- **카카오 REST API** — 한국 블로그·카페 반응 수집

## 왜 "복사 → 붙여넣기" 방식인가?

네이버 블로그는 글 등록 공식 API가 종료되어(2019), 완전 무인 등록은 비공식 자동화(계정 제재 위험)뿐입니다.
그래서 이 도구는 **글 작성에 걸리는 시간을 자동화**하는 데 집중합니다 — 생성부터 발행까지 1~2분이면 충분합니다.
인스타그램 역시 개인 계정은 API 게시가 불가하므로 이미지 자동 생성 + 캡션 원클릭 복사 방식을 사용합니다.
